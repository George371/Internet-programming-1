About Page

This application is a basic web server built using Express.js, designed to demonstrate understanding of the fundamentals of server-side development. This application have 3 routes: a home page, an about page, and a contact page, with form submission and logging data to console.



  1. Home Page (/): Displays a welcoming message to users. Simple HTML response to greet visitors.
  2. About Page (/about): Provides information about the application. Includes a brief description stating that it is a simple Express.js application.
  3. Contact Page (/contact): Renders an HTML form allowing users to submit their name and email address. The form uses a POST request to send data back to the server.
